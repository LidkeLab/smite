## Documentation Conventions

### Code Blocks

All MATLAB code uses triple-backtick code blocks with `matlab` language tag:

```matlab
% Working example code
SMF = smi_core.SingleMoleculeFitting();
```

### File Paths

Absolute paths in examples use forward slashes (cross-platform):
```matlab
SMF.Data.FileDir = '/path/to/data';
```

Windows users: Replace with backslashes or use MATLAB path functions.

### Units

Explicit units are always specified:
- Distances: pixels, micrometers (Î¼m), nanometers (nm)
- Time: seconds (s), frames
- Photons: photons or photons/pixel
- Frame rate: Hz (frames per second)

### Cross-References

Internal links use relative paths:
```markdown
[SMF Structure](core-concepts/smf-structure.md)
```
